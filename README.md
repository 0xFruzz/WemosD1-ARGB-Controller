# ESP8266 (Wemos D1) LED Controller

Прошивка для управления адресной светодиодной лентой (WS2812B) через Wi-Fi. Позволяет менять цвет и яркость с помощью HTTP-запросов и сохраняет настройки в EEPROM.

## Install
1. Откройте код в Arduino IDE.
2. Установите библиотеку **FastLED**.
3. Введите название вашей Wi-Fi сети и пароль в переменные `ssid` и `password`.
4. В строке `#define NUM_LEDS` укажите количество светодиодов в вашей ленте.
5. Прошейте плату.

## REST-API
С данной прошивкой Wemos становится вебсервером, запросы можно отправлять как с браузера, так и с любого HTTP Клиента

### 1. Проверка состояния
**Запрос:** `GET http://[IP_АДРЕС]/stats`  
**Ответ:** JSON с текущими значениями яркости и цвета.

### 2. Изменение настроек
**Запрос:** `GET http://[IP_АДРЕС]/setup?r=[0-255]&g=[0-255]&b=[0-255]&brightness=[0-255]`

**Примеры:**
* **Включить красный цвет:** `http://192.168.x.x/setup?r=255&g=0&b=0`
* **Установить яркость на 50%:** `http://192.168.x.x/setup?brightness=128`
* **Зеленый цвет на минимальной яркости:** `http://192.168.x.x/setup?r=0&g=255&b=0&brightness=10`

## EEPROM
Устройство автоматически сохраняет настройки после каждого изменения. При перезагрузке или сбое питания лента включится с тем же цветом и яркостью, которые были установлены последними.
